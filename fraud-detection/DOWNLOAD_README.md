### 🚀 How to Use After Download:

#### Step 1: Extract
```bash
tar -xzf fraud-detection-system-DOWNLOAD.tar.gz
cd fraud-detection-system
```

#### Step 2: Setup Backend
```bash
cd backend
pip install -r requirements.txt
```

#### Step 3: Setup Frontend
```bash
cd frontend
yarn install
```

#### Step 4: Configure
- Edit `backend/.env` for MongoDB connection
- Edit `frontend/.env` for backend URL

#### Step 5: Start MongoDB
```bash
# Install MongoDB first if not installed
# Then run:
mongod --dbpath /path/to/data
```

#### Step 6: Run
```bash
# Terminal 1 - Backend
cd backend
uvicorn server:app --host 0.0.0.0 --port 8001

# Terminal 2 - Frontend
cd frontend
yarn start
```

#### Step 7: Access
Open browser to `http://localhost:3000`
